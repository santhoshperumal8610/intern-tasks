# Responsive Landing Page

## Project Overview
This project is a simple and responsive landing page built using HTML and CSS. It includes a header with navigation links, a hero section with a call-to-action button, and a footer with social media links. The page is designed to be responsive and works well on both desktop and mobile devices.

## Features
- Responsive design using CSS media queries
- Header with logo and navigation menu
- Hero section with heading, description, and call-to-action button
- Footer with social media links
- Clean and modern user interface
- Built using HTML5 and CSS3
- Flexbox layout for responsive alignment

## Technologies Used
- HTML5
- CSS3
- Visual Studio Code
- Live Server Extension

## Project Structure
```
LandingPage/
│── index.html
│── style.css
└── README.md
```

## How to Run the Project
1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Install the **Live Server** extension (if not already installed).
4. Right-click on `index.html` and select **Open with Live Server**.
5. The landing page will open in your default web browser.

## Responsive Design
The website is fully responsive and adapts to different screen sizes using CSS media queries.
